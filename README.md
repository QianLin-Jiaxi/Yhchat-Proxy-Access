# Yhchat-Proxy-Access
#### 为了能绕过云湖防盗链，提供给开发者一个可以自主使用云湖图床的机会，故此制作开发项目使用此功能可实现绕过云湖防盗链直接访问数据文件
#### 我们公开了脚本文件，若有需要的开发者可以复制后单独部署
#### 若图床被墙或出现故障请联系我
> **访问URL:** https://yhapi.qianlinjiaxi.qzz.io/yhproxy.php  
> **type参数:** key所处的数据分区，audio-音频，file-文件，image-图片，video-视频  
> **key参数:** 数据上传返回的key  
> **功能链接实例:** https://yhapi.qianlinjiaxi.qzz.io/yhproxy.php?type=image&key=bf8eea8cbc31b1ea1a5b5424a20f92c2.jpg  
> **云湖原始链接实例:** https://chat-img.jwznb.com/bf8eea8cbc31b1ea1a5b5424a20f92c2.jpg  
#### 切勿使用此功能进行非法活动！后果由使用此者承担
